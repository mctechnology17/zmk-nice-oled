## New Variables (v0.0.3)

| Variable | Type | Default | Description |
|----------|------|---------|-------------|
| `CONFIG_ZMK_DISPLAY_BLANK_ON_IDLE` | bool | `n` (OLED), `y` (ePaper) | Control display blanking on idle for nice_custom/nice_epaper |
| `CONFIG_NICE_OLED_WIDGET_MODIFIERS_INDICATORS_BONGO_CAT` | bool | `n` | Enable Bongo Cat animation for Modifiers Indicators |
| `CONFIG_NICE_OLED_WIDGET_MODIFIERS_INDICATORS_BONGO_CAT_ANIMATION_MS` | int | `300` | Animation speed (ms) for Modifiers Bongo Cat |
| `CONFIG_NICE_OLED_WIDGET_HID_INDICATORS_BONGO_CAT` | bool | `n` | Enable Bongo Cat animation for HID Indicators |
| `CONFIG_NICE_OLED_WIDGET_HID_INDICATORS_BONGO_CAT_ONLY_CAPSLOCK` | bool | `y` | Show HID Bongo Cat only when CapsLock is active |
| `CONFIG_NICE_OLED_WIDGET_HID_INDICATORS_BONGO_CAT_ANIMATION_MS` | int | `300` | Animation speed (ms) for HID Indicators Bongo Cat |
