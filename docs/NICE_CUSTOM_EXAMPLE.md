# Nice Custom - Widget Position Configuration

Example configuration for `nice_custom` shield with all customizable X/Y coordinate variables.

## Device Tree Overlay (nice_custom.overlay)

```dts
/*
 * Copyright (c) 2022 The ZMK Contributors
 *
 * SPDX-License-Identifier: MIT
 */

&nice_view_spi {
    status = "okay";
    nice_view: ls0xx@0 {
        compatible = "sharp,ls0xx";
        spi-max-frequency = <1000000>;
        reg = <0>;
        width = <160>;   // Your display width, example like nice!view
        height = <68>;  // Your display height, example like nice!view
    };
};

/ {
    chosen {
        zephyr,display = &nice_view;
    };
};
```

## Example 2: RAW HID Configuration (400x240 Display)

Complete example with RAW HID enabled for host communication (time, layout, volume, weather, media player).

### nice_custom.overlay

```dts
/*
 * Copyright (c) 2022 The ZMK Contributors
 *
 * SPDX-License-Identifier: MIT
 */

&nice_view_spi {
    status = "okay";
    nice_view: ls0xx@0 {
        compatible = "sharp,ls0xx";
        spi-max-frequency = <1000000>;
        reg = <0>;
        width = <400>; // example with a big display
        height = <240>; // example with a big display
    };
};

/ {
    chosen {
        zephyr,display = &nice_view;
    };
};
```

### RAW HID Variables Reference

| Variable | Default | Description |
|----------|---------|-------------|
| `CONFIG_NICE_OLED_WIDGET_RAW_HID` | n | Enable RAW HID feature |
| `CONFIG_NICE_OLED_WIDGET_RAW_HID_DRIVER` | y | Enable RAW HID driver |
| `CONFIG_USB_HID_DEVICE_COUNT` | 2 | Number of HID devices |
| `CONFIG_NICE_OLED_WIDGET_RAW_HID_USAGE_PAGE` | 0xFF60 | HID Usage Page |
| `CONFIG_NICE_OLED_WIDGET_RAW_HID_USAGE` | 0x61 | HID Usage |
| `CONFIG_NICE_OLED_WIDGET_RAW_HID_REPORT_SIZE` | 32 | Report size in bytes |
| `CONFIG_NICE_OLED_WIDGET_RAW_HID_DEVICE` | "HID_1" | HID device name |
| `CONFIG_NICE_OLED_WIDGET_RAW_HID_TIME` | y | Show time from host |
| `CONFIG_NICE_OLED_WIDGET_RAW_HID_LAYOUT` | y | Show keyboard layout |
| `CONFIG_NICE_OLED_WIDGET_RAW_HID_VOLUME` | y | Show system volume |
| `CONFIG_NICE_OLED_WIDGET_RAW_HID_WEATHER` | n | Show weather/temperature |
| `CONFIG_NICE_OLED_WIDGET_RAW_HID_MEDIA_PLAYER_SPOTIFY_MACOS` | n | Show Spotify info (macOS) |
| `CONFIG_NICE_OLED_WIDGET_RAW_HID_LAYOUT_LIST` | "DE,US,LA,AB" | Layout names |

> **Note**: RAW HID requires a companion app on the host computer to send data to the keyboard.
